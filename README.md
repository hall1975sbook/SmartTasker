# SmartTasker

![SmartTasker Logo](https://via.placeholder.com/150)

## Overview

SmartTasker is an innovative task management application that leverages artificial intelligence to help users organize, prioritize, and automate their daily tasks seamlessly. Designed for productivity enthusiasts, SmartTasker learns from your habits and suggests the best way to manage your workload effectively.

## Features

- AI-driven task prioritization and scheduling
- Natural language task entry
- Automated reminders and notifications
- Integration with popular calendar apps
- Customizable task categories and tags
- Dark and light mode support

## Tech Stack

- Frontend: React.js, Tailwind CSS
- Backend: Node.js, Express
- AI Integration: TensorFlow.js
- Database: MongoDB
- Authentication: JWT

## Installation

1. Clone the repository:

   ```bash
   git clone https://github.com/hall1975sbook/SmartTasker.git
   cd SmartTasker
   ```

2. Install dependencies for backend and frontend:

   ```bash
   cd backend
   npm install
   cd ../frontend
   npm install
   ```

3. Configure environment variables:

   Create a `.env` file in the `backend` directory with the following keys:

   ```ini
   MONGO_URI=your_mongodb_connection_string
   JWT_SECRET=your_jwt_secret_key
   ```

4. Start the backend server:

   ```bash
   cd backend
   npm start
   ```

5. Start the frontend application:

   ```bash
   cd ../frontend
   npm start
   ```

## Usage

- Open your browser and navigate to `http://localhost:3000`.
- Sign up or log in using your credentials.
- Add tasks using natural language (e.g., "Finish report by Friday").
- Let SmartTasker prioritize your tasks and set reminders.
- Explore integrations to sync with your calendar apps.
